# React Native Memory Game App. 

✔ Find the same items.
✔ Collect points.
✔ Reset Game.


## Technologies ✅ CSS ⚛ React Native ✅ Expo

Download the repository and run it locally:

### `npm install`

### `expo start`

Runs the app in the Google Play app Expo Go. Scan QR code.<br />
